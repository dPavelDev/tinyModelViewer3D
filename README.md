# tinyModelViewer3D

Группа: 553501

Разработчик: Дорошкевич Павел Евгеньевич

Тема: Масштабирование и вращение 3D объектов.


### Описание проекта: 

1) Вращение и масштабирование 3D модели (куб)

2) Используемые библиотеки: Qt 5.5

3) Поддерживаемые системы: Windows, Linux


### План развития проекта:

1) Добавление различный объектов для отображения

2) Возможность загрузки объектов из файла (формат *.obj)

3) Оптимизация рендера для плавного отображения высокополигональных объектов
